# 🏙️ City Heist

A GTA 2–style top-down 3D driving game that runs entirely in the browser (no install, no internet needed to play).

Built with [Three.js](https://threejs.org/). Single self-contained `index.html` — just open it.

## 🎮 Play

Open `index.html` in any modern browser. Or visit the live version (GitHub Pages):

> 👉 **[play the game](https://plilian.github.io/city-heist/)**

## 🕹️ Controls

| Key | Action |
|-----|--------|
| `W A S D` | Move / Drive |
| `Shift` | Sprint |
| `F` | Enter / Exit car |
| `Space` | Handbrake |

## 🎯 Mission

Steal the yellow sports car (marked) and deliver it to the green garage across the city.
